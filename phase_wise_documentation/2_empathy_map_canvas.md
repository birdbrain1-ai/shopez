# Ideation Phase
## Empathize & Discover (Empathy Map Canvas)
* **Date**: 18 June 2026
* **Project Name**: Shopez

### Empathy Map Canvas
An empathy map captures knowledge about a user’s behaviors and attitudes to help the team better understand them.

#### Target User Persona: Retail Customer (Shopper) & Admin Manager
| Quadrant | What the User Thinks and Feels | What the User Hears |
|---|---|---|
| **SAY & DO** | • Wants to check out quickly without friction.<br>• Looks for clear product lists and pricing.<br>• Clicks directly on demo credentials during testing. | • "Make sure your online payment is secure!"<br>• "I prefer clean, simple light-themed designs." |
| **THINK & FEEL** | • Fears entering card details on unsafe forms.<br>• Appreciates micro-animations (like card flipping).<br>• Wants real-time order status confirmations. | • "Many e-commerce sites feel cluttered and slow."<br>• Friends recommending quick e-shops. |
| **SEE** | • Clean product grids with high-quality images.<br>• Visual card graphics rendering dynamically.<br>• Interactive dashboard stats reflecting store sales. | • Standard modern minimalist e-commerce web layouts.<br>• Instant feedback notices (green success alerts). |
| **PAIN & GAIN** | • **Pain**: White text conflicts, server downtime, complex database setups.<br>• **Gain**: Local DB fallback ensures the site is always interactive. | • **Pain**: White-on-white text issues on admin insights.<br>• **Gain**: Real-time sales insights in black text. |
